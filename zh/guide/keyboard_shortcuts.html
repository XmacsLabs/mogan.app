墨干的键盘快捷键 ​

欢迎查看墨干的键盘快捷键指南。这里列出了可用快捷键组合，帮助您更高效地使用软件。

文档来自 Liii STEM 官网。该官网的快捷键列表会持续更新，墨干通常会及时同步更新内容，但可能存在一定延迟，请以官方文档为准。您也可在官网获取 PDF 版本的最新快捷键手册。

环境快捷键 ​

提示

这些快捷键用于控制文档的整体环境和格式设置。包括空格、缩进、对齐方式、章节标题、列表等基本排版功能。正确使用这些快捷键可以帮助您更高效地进行文档编辑。

WindowsGNU/LinuxMac等价说明
space + tabspace + tabspace + tabNon-breaking space ( or )插入不间断空格，保持文字之间的固定间距
ctrl + t//段落首行缩进
ctrl + l//将文本右对齐
ctrl + e//将文本居中对齐
ctrl + r//将文本左对齐
alt + 1alt + 1option + 1创建一级标题
alt + 2alt + 2option + 2创建二级标题
alt + 3alt + 3option + 3创建三级标题
alt + 4alt + 4option + 4创建段落标题
+ + tab+ + tab+ + tab创建无序列表项
1 + . + tab1 + . + tab1 + . + tab创建有序列表项
$$$inline math mode插入行内数学公式
alt + $alt + $option + $single-line math mode插入单行独立数学公式
alt + 7alt + 7option + 7multi-line math mode插入多行数学公式环境
ctrl + #ctrl + #ctrl + #add equation number为数学公式添加编号
alt + arrowalt + arrowoption + arrowadd new row/column in matrix/table在矩阵或表格中添加新的行或列
ctrl + shift + fctrl + shift + fctrl + shift + fadd footnote添加脚注
ctrl + nctrl + nctrl + nadd new script创建新文档
ctrl + pctrl + pctrl + pexport to PDF将文档导出为PDF格式

注意事项

  1. Mac系统下使用 option 键代替 Windows/Linux 下的 alt 键
  2. 某些快捷键可能与系统快捷键冲突，需要在系统设置中更改
  3. 数学公式相关的快捷键需要在相应的数学环境中使用
  4. 表格和矩阵的操作快捷键仅在对应环境中生效
  5. 建议熟记常用快捷键以提高编辑效率
  6. 段落首行缩进、右对齐、居中对齐、左对齐快捷键暂只支持Windows使用

装饰符号 ​

提示

在Liii STEM中，您可以使用键盘快捷键快速输入常用的数学装饰符号。这些符号通常用于表示导数、向量等数学概念。

Windows/GNU/LinuxMac等价说明
alt + ˙option + ˙ (\dot{})一阶导数符号
alt + ˙ + ˙option + ˙ + ˙ (\ddot{})二阶导数符号
alt + ˆoption + ˆ (\hat{})单位向量符号
alt + ˜option + ˜ (\tilde{})等价符号
alt + -option + - (\bar{})平均值符号

注意事项

  1. 装饰符号需要在字符输入后才能添加
  2. 某些装饰符号可能与系统快捷键冲突，请确保您的系统设置允许这些快捷键
  3. Mac用户请确保已启用option键的特殊字符输入功能

括号与定界符 ​

提示

Liii STEM提供了多种括号和定界符的快捷输入方式。通过组合键可以快速输入常用的数学符号，提高公式编辑效率。

  3. 建议先熟悉常用的快捷键组合，逐步掌握更多组合
