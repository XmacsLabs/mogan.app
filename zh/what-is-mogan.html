什么是墨干 ​

享受探索科学与技术的乐趣！

墨干理工套件（Mogan STEM）是墨干实验室旗下永久免费的结构化理工套件，专为科学探索和技术创新而设计。它是一个面向数学、物理、统计与计算机科学的专业科研写作平台，专门用于处理复杂公式类文档。它在 GNU TeXmacs 的基础上进行了深度优化，着重于性能与用户体验。使用墨干，可以比使用 快 100 倍地创建科学文档！

访问官方网站：

  • 墨干（社区版）: https://mogan.app
  • Liii STEM（商业版，内置 AI）: https://liiistem.com

墨干Mogan STEM永久免费的结构化科技编辑器平台成熟

作为GNU TeXmacs的发行版之一，墨干在继承其优秀基因的基础上，专注于为理工科用户提供更加流畅、高效的文档创作体验。

主要特性 ​

墨干理工套件提供了直观的图形界面，让您能够轻松创建高质量的技术文档，达到与LaTeX相媲美的专业水准：

📋 魔法粘贴（Magic Paste） ​

  • 支持 ChatGPT、Claude、Gemini 等主流 AI 模型
  • 自动识别数学公式、代码块、表格及其他复杂格式
  • 一键转换格式，无需手动调整
  • 保持原始格式与样式完整性

🛠 导出 PDF 与 LaTeX ​

  • 支持多种格式的批量导出，满足不同使用场景
  • PDF 导出：保持原始文档的版面效果，支持高分辨率打印
  • LaTeX 导出：提高编译通过率，兼容主流期刊投稿要求

⚡ 快速数学编辑 ​

  • 独立数学模式：快速切换，专注公式编辑
  • 智能快捷键：乐高式记忆规则，Tab 键快速循环，减少鼠标操作
  • 丰富的符号库：覆盖完整的数学与物理符号，配合大量按键组合

🎓 零基础上手，论文级编辑 ​

  • 三大模式无缝切换，操作简单，上手快，更高效
  • 不同内容类型结构清晰，文章层级组织良好
  • 点击插入环境，复杂排版变得简单

科学文档创作 ​

  • 统一编辑体验：为文本、数学公式、图像、交互式内容和幻灯片等不同类型的结构化文档提供一致且优化的编辑体验
  • 强大的扩展性：可作为各种科学计算系统的图形化前端，包括计算机代数系统、数值分析系统、统计分析系统等
  • 专业输出质量：内置高质量排版算法，可输出适合打印和屏幕展示的专业文档
  • 高质量排版：专业的文档格式，输出近 质量的排版效果
  • 数学符号：支持高级公式编辑并兼容
  • 参考文献管理：内置参考与引用管理工具

性能与可用性 ​

  • 更快启动：相较 TeXmacs 优化了软件性能和大量的交互方式
  • 响应式界面：基于现代 Qt6 的用户界面
  • 跨平台支持：支持Windows、macOS和所有主流Unix平台
  • 自定义功能：支持使用Scheme扩展语言为编辑器添加新功能，自定义排版样式

与其余科技工具的交互 ​

  • 插件系统：可通过多种编程语言（Python, C++, Scheme）与工具扩展功能
  • 文档格式：支持 TeXmacs、LaTeX、PDF、HTML 等格式
  • 交互绘图：与数据分析工具集成实现交互式绘图

安装 ​

墨干（社区版） ​

从 Release 下载：从 GitHub releases 下载针对你平台的最新发行版本。

从源码构建：

bash
# 克隆仓库
git clone https://github.com/XmacsLabs/mogan.git
cd mogan

# 使用 xmake 构建 stem
xmake build stem

如需带 AI 能力的商业版，请访问 Liii STEM 官方网站 获取安装选项。

墨干得名于家乡的避暑胜地——莫干山。犹忆青春年少时，在高中同桌老家，靠近莫干山的一个小山村里面，住过一晚上。傍晚，我们就在屋后小溪里面洗澡。清晨，我们跟着他在山上兜兜转转挖笋。中午，一起吃刚挖好的笋。溪水潺潺，篁竹幽幽，炊烟袅袅……

发展历程 ​

「墨干编辑器」的诞生（2022年）

沈浪熊猫儿于2022年1月30号，基于Massimiliano Gubinelli的S7 Scheme分支，发布了墨干编辑器v1.0.0，并创建了墨客实验室。

重定位为「墨干理工套件」（2023年）

从V1.2.0开始，我们将墨干编辑器重命名并重定位为墨干理工套件，致力于为理工科用户提供更专业、更高效的文档创作工具。

技术背景 ​

墨干基于GNU TeXmacs开发，继承了其优秀的结构化编辑和高质量排版特性，同时修复了许多错误，优化了性能，并美化了界面。关于GNU TeXmacs的更多信息，请参考GNU TeXmacs主页。
