The Maxima session plugin relies on the Ghostscript Binary plugin. Please follow the documentation to install Ghostscript first, and then check Help -> Plugins -> Maxima to ensure that the PDF images in the documentation can be rendered correctly.

Download and install from the Maxima official website.

The recommended version of Maxima is 5.47.0.

Linux â€‹

On Debian and its derivatives, use the following command to install:
