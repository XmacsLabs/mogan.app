Debian 12 â€‹

Download the official deb and then:

Ubuntu 20.04/Ubuntu 22.04 â€‹

Download the official deb and then

sudo apt install ./mogan-research-v1.2.5.3-ubuntu20.04.deb

# For ubuntu 22.04
sudo apt install ./mogan-research-v1.2.5.3-ubuntu22.04.deb
